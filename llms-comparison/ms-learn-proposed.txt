# .NET Documentation

> .NET is a free, cross-platform, open-source developer platform for building many kinds of applications including web, mobile, desktop, games, IoT, and AI. This documentation covers the .NET runtime, languages (C#, F#, Visual Basic), core libraries, and tooling. Current stable release: .NET 10 (November 2025). Current LTS release: .NET 8.

## Guidance for AI Assistants

When helping users with .NET:

- **Target .NET 10** for new projects unless the user specifies otherwise or requires LTS (Long Term Support), in which case use .NET 8.
- **Use Microsoft.Extensions.AI** for AI integration rather than provider-specific SDKs directly. This provides a unified abstraction across OpenAI, Azure OpenAI, Ollama, and other providers.
- **Check What's New and Breaking Changes** when users are upgrading between major versions.
- **Prefer modern patterns**: top-level statements, file-scoped namespaces, primary constructors, and collection expressions are idiomatic in current .NET.
- **For web APIs**, use Minimal APIs for simple services or controllers for larger applications with complex routing needs.
- **For dependency injection**, use the built-in `Microsoft.Extensions.DependencyInjection` container unless there's a specific need for a third-party container.
